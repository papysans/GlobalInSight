## 概述

数据洞察页面展示舆情分析的多维可视化结果，包含以下核心组件：

| 组件 | 数据类型 | 来源 |
|------|---------|------|
| 雷达图 | 真实数据 ✅ | 后端 Crawler 平台统计 |
| 辩论时间线 | 真实数据 ✅ | Analyst Agent 多轮输出 |
| 热度趋势图 | AI 推演 ⚠️ | 基于日志内容的语义分析 |
| 洞察卡 | 真实数据 ✅ | Analyst 解析结果 |
| 关键发现 | 真实数据 ✅ | 洞察内容分句提取 |

## 1. 雷达图（平台覆盖分布）

### 数据来源
- **真实数据** ✅
- 来自后端 `Crawler` Agent 返回的 `platform_stats` 字段
- 包含每个平台实际爬取到的内容数量

### 数据格式
```json
{
  "wb": 15,      // 微博 15 条
  "bili": 8,    // B站 8 条
  "xhs": 12,    // 小红书 12 条
  "zhihu": 6    // 知乎 6 条
}
```

### 计算逻辑
```javascript
// 归一化到 50-100 范围
const maxCount = Math.max(...Object.values(platformStats));
const normalized = 50 + (count / maxCount) * 50;
```

### 代码位置
- Store: `src/stores/analysis.js` → `radarChartData` getter
- 组件: `src/components/RadarChart.vue`

## 2. 辩论时间线（推理演化过程）

### 数据来源
- **真实数据** ✅
- 来自 `Analyst` Agent 的多轮输出日志
- 每轮输出包含结构化字段

### 数据格式
```
INSIGHT: 这一事件折射出...
TITLE: 尼帕病毒警戒升级
SUB: 高致死病毒 筑牢防控防线
SUMMARY: 精炼的一句话观点
```

### 计算逻辑
```javascript
// 过滤 Analyst 日志
const analystLogs = logs.filter(log => log.agent_name === 'Analyst');

// 正则提取结构化字段
const titleMatch = content.match(/TITLE:\s*(.+?)(?=\s*(?:SUB:|INSIGHT:|$))/is);
const summaryMatch = content.match(/SUMMARY:\s*(.+?)(?=\s*(?:TITLE:|$))/is);
```

### 代码位置
- Store: `src/stores/analysis.js` → `debateTimelineData` getter
- 组件: `src/components/DebateTimeline.vue`
- Canvas: `src/components/canvas/DebateTimelineCanvas.vue`

## 3. 热度趋势图

### 数据来源
- **AI 推演模型** ⚠️
- 基于 `Analyst` 日志的**内容长度**和**热度关键词密度**计算
- **不是**真实的时间序列热度数据

### 热度关键词
```javascript
const heatKeywords = ['热议', '关注', '讨论', '争议', '热度', '爆发', '火爆', '刷屏'];
```

### 计算逻辑
```javascript
// 1. 计算每轮热度评分
const lengthScore = Math.min(contentLength / 10, 50);  // 最多50分
const keywordScore = keywordCount * 10;                // 每个关键词10分
const roundHeat = Math.min(lengthScore + keywordScore, 100);

// 2. 生成7点趋势曲线（插值）
// - 单轮：模拟爆发曲线 [30%, 50%, 70%, 100%, 95%, 90%, 85%]
// - 多轮：线性插值生成平滑曲线

// 3. 判断阶段
if (growth > 100) stage = '爆发期';
else if (growth < -20) stage = '回落期';
else if (avgHeat > 80) stage = '高热期';
else stage = '扩散期';
```

### 代码位置
- Store: `src/stores/analysis.js` → `trendChartData` getter
- 组件: `src/components/TrendChart.vue`
- Canvas: `src/components/canvas/TrendChartCanvas.vue`

## 4. 洞察卡（InsightCard）

### 数据来源
- **真实数据** ✅

### 字段说明
| 字段 | 来源 | 说明 |
|------|------|------|
| `conclusion` | Analyst INSIGHT 字段 | 核心洞察结论 |
| `platforms` | 用户选择 | 选中的平台数量 |
| `debateRounds` | Analyst 日志数 | 辩论轮数 |
| `controversy` | Debater 日志数 | 争议程度（高/中/低） |

### 争议程度计算
```javascript
const critiqueCount = logs.filter(log => log.agent_name === 'Debater').length;
const controversy = critiqueCount > 3 ? '高' : critiqueCount > 1 ? '中' : '低';
```

### 代码位置
- Store: `src/stores/analysis.js` → `insightCardData` getter
- 组件: `src/components/InsightCard.vue`

## 5. 关键发现

### 数据来源
- **真实数据** ✅
- 从 `insight` 字段按句号分割，取前3条

### 计算逻辑
```javascript
const sentences = insight.split(/[。！？]/).filter(s => s.trim().length > 10);
return sentences.slice(0, 3);
```
